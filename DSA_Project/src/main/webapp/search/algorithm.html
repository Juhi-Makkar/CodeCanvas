1. Linear Search (Sequential Search)

Search for target by sequentially checking each element from start to end.

Time: O(n) Space: O(1) Worst: O(n) Best: O(1)

Algorithm Steps

1
Start from Beginning
Set index i = 0
2
Compare Elements
If arr[i] == target, return i
3
Continue Search
Increment i, repeat until end
4
Not Found
Return -1 if target not found

Pseudocode

PROCEDURE linearSearch(arr, target, n)
    FOR i = 0 TO n-1 DO
        IF arr[i] == target THEN
            RETURN i
        END IF
    END FOR
    RETURN -1
END PROCEDURE

Implementation

C int linearSearch(int arr[], int n, int target) { for(int i = 0; i < n; i++) { if(arr[i] == target) { return i; } } return -1; }

2. Linear Search (Sentinel Approach)

Optimized linear search by placing target at end as sentinel.

Time: O(n) Space: O(1)

Pseudocode

PROCEDURE sentinelLinearSearch(arr, n, target)
    temp = arr[n-1]
    arr[n-1] = target
    i = 0
    WHILE arr[i] != target DO
        i = i + 1
    END WHILE
    arr[n-1] = temp
    IF i < n THEN
        RETURN i
    ELSE
        RETURN -1
    END IF
END PROCEDURE

Implementation

C int sentinelSearch(int arr[], int n, int target) { int last = arr[n-1]; arr[n-1] = target; int i = 0; while(arr[i] != target) i++; arr[n-1] = last; return (i < n-1) ? i : -1; }

3. Binary Search (Iterative)

Efficient search on sorted array by repeatedly dividing search interval in half.

Time: O(log n) Space: O(1) Worst: O(log n) Best: O(1)

Algorithm Steps

1
Initialize Bounds
low = 0, high = n-1
2
Calculate Mid
mid = (low + high) / 2
3
Compare & Adjust
Adjust low/high based on comparison
4
Repeat/Terminate
Continue until found or low > high

Pseudocode

PROCEDURE binarySearch(arr, target, n)
    low = 0
    high = n - 1
    WHILE low <= high DO
        mid = (low + high) / 2
        IF arr[mid] == target THEN
            RETURN mid
        ELSE IF arr[mid] < target THEN
            low = mid + 1
        ELSE
            high = mid - 1
        END IF
    END WHILE
    RETURN -1
END PROCEDURE

Implementation

C int binarySearch(int arr[], int n, int target) { int low = 0, high = n - 1; while(low <= high) { int mid = low + (high - low) / 2; if(arr[mid] == target) return mid; else if(arr[mid] < target) low = mid + 1; else high = mid - 1; } return -1; }

4. Binary Search (Recursive)

Recursive implementation of binary search algorithm.

Time: O(log n) Space: O(log n)

Pseudocode

FUNCTION binarySearchRecursive(arr, target, low, high)
    IF low > high THEN
        RETURN -1
    END IF
    
    mid = (low + high) / 2
    
    IF arr[mid] == target THEN
        RETURN mid
    ELSE IF arr[mid] < target THEN
        RETURN binarySearchRecursive(arr, target, mid+1, high)
    ELSE
        RETURN binarySearchRecursive(arr, target, low, mid-1)
    END IF
END FUNCTION

Implementation

C int binarySearchRec(int arr[], int target, int low, int high) { if(low > high) return -1; int mid = low + (high - low) / 2; if(arr[mid] == target) return mid; else if(arr[mid] > target) return binarySearchRec(arr, target, low, mid - 1); else return binarySearchRec(arr, target, mid + 1, high); }

5. Binary Search First/Last Occurrence

Find first and last occurrence of target in sorted array with duplicates.

Time: O(log n) Space: O(1)

Implementation

C int firstOccurrence(int arr[], int n, int target) { int low = 0, high = n - 1, result = -1; while(low <= high) { int mid = low + (high - low) / 2; if(arr[mid] == target) { result = mid; high = mid - 1; // Continue searching left } else if(arr[mid] < target) low = mid + 1; else high = mid - 1; } return result; }

6. Jump Search

Jump search performs better than linear search on sorted arrays by jumping ahead by fixed steps.

Time: O(√n) Space: O(1)

Algorithm Steps

1
Calculate Jump Size
step = √n
2
Jump Forward
Jump by step size until overshoot
3
Linear Search
Linear search in final block

Implementation

C int jumpSearch(int arr[], int n, int target) { int step = sqrt(n); int prev = 0; while(arr[min(step, n)-1] < target) { prev = step; step += sqrt(n); if(prev >= n) return -1; } // Linear search in block while(prev < min(step, n) && arr[prev] < target) prev++; if(prev < n && arr[prev] == target) return prev; return -1; }

7. Interpolation Search

Interpolation search estimates position based on value distribution (works on uniformly distributed data).

Time: O(log log n) avg Space: O(1) Worst: O(n)

Implementation

C int interpolationSearch(int arr[], int n, int target) { int low = 0, high = n - 1; while(low <= high && target >= arr[low] && target <= arr[high]) { if(low == high) { if(arr[low] == target) return low; return -1; } int pos = low + ((double)(high - low) * (target - arr[low]) / (arr[high] - arr[low])); if(arr[pos] == target) return pos; if(arr[pos] < target) low = pos + 1; else high = pos - 1; } return -1; }

8. Exponential Search

Exponential search finds range first, then performs binary search (good for unbounded arrays).

Time: O(log n) Space: O(1)

Implementation

C int exponentialSearch(int arr[], int n, int target) { if(arr[0] == target) return 0; int i = 1; while(i < n && arr[i] <= target) i *= 2; return binarySearch(arr, i/2, min(i, n), target); }
