📈 Search Fundamentals

Master Linear Search (O(n)), Binary Search (O(log n)), and advanced search techniques for interviews.

// Binary Search Template int binarySearch(int arr[], int n, int target) { int left = 0, right = n - 1; while (left <= right) { int mid = left + (right - left) / 2; if (arr[mid] == target) return mid; else if (arr[mid] < target) left = mid + 1; else right = mid - 1; } return -1; }

Key Algorithms

  • Linear Search: O(n) - Simple, unsorted arrays
  • Binary Search: O(log n) - Sorted arrays only
  • Two Pointers: O(n) - Sorted arrays, pairs
  • Sliding Window: O(n) - Subarrays/Strings
  • HashMap Search: O(1) average - Existence/Uniqueness
