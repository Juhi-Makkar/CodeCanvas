1. Linear Search (Sequential Search)

Linear search checks each element sequentially until the target is found or end is reached.

Best Case: O(1) - Element at first position
Worst Case: O(n) - Element at last position or not found
Average Case: O(n/2)

Basic Implementation

C int linearSearch(int arr[], int n, int key) { for(int i = 0; i < n; i++) { if(arr[i] == key) return i; } return -1; }

With Found Flag

C int linearSearch(int arr[], int n, int key) { int found = 0; for(int i = 0; i < n && !found; i++) { if(arr[i] == key) { found = 1; return i; } } return -1; }

Usage Example

C int arr[] = {64, 34, 25, 12, 22, 11, 90}; int n = 7; int key = 22; int result = linearSearch(arr, n, key); if(result != -1) printf("Found at index %d", result); else printf("Not found");

2. Binary Search

Binary search works on sorted arrays by repeatedly dividing search interval in half.

Prerequisite: Array must be sorted!
Best Case: O(1) - Middle element
Worst Case: O(log n)
Average Case: O(log n)

Iterative Implementation

C int binarySearch(int arr[], int n, int key) { int low = 0, high = n - 1; while(low <= high) { int mid = low + (high - low) / 2; if(arr[mid] == key) return mid; else if(arr[mid] < key) low = mid + 1; else high = mid - 1; } return -1; }

Recursive Implementation

C int binarySearchRec(int arr[], int low, int high, int key) { if(low > high) return -1; int mid = low + (high - low) / 2; if(arr[mid] == key) return mid; else if(arr[mid] < key) return binarySearchRec(arr, mid + 1, high, key); else return binarySearchRec(arr, low, mid - 1, key); }

Usage Example

C // Sorted array int arr[] = {2, 3, 4, 10, 40}; int n = 5, key = 10; int result = binarySearch(arr, n, key);

3. Jump Search

Jump search is efficient for sorted arrays. It jumps ahead by fixed steps, then performs linear search.

Best Case: O(1)
Worst Case: O(√n)
Optimal Block Size: √n

Implementation

C int jumpSearch(int arr[], int n, int key) { int step = sqrt(n); int prev = 0; while(arr[min(step, n)-1] < key) { prev = step; step += sqrt(n); if(prev >= n) return -1; } // Linear search in block while(arr[prev] < key && prev < n) { prev++; if(arr[prev] == key) return prev; } return -1; }

4. Interpolation Search

Interpolation search estimates the position of target using interpolation formula. Works best for uniformly distributed values.

Best Case: O(1) - Uniform distribution
Worst Case: O(n) - Skewed distribution
Average Case: O(log log n)

Implementation

C int interpolationSearch(int arr[], int n, int key) { int low = 0, high = n - 1; while(low <= high && key >= arr[low] && key <= arr[high]) { if(low == high) { if(arr[low] == key) return low; return -1; } // Interpolation formula int pos = low + ((double)(high - low) * (key - arr[low]) / (arr[high] - arr[low])); if(arr[pos] == key) return pos; else if(arr[pos] < key) low = pos + 1; else high = pos - 1; } return -1; }
