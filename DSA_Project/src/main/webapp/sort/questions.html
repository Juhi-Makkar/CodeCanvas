Master comparison & non-comparison sorting. Understand time/space complexity, stability, and adaptive properties.

// Bubble Sort - C void bubbleSort(int arr[], int n) { for(int i = 0; i < n-1; i++) for(int j = 0; j < n-i-1; j++) if(arr[j] > arr[j+1]) swap(&arr[j], &arr[j+1]); }

Time & Space Complexity

Bubble Sort
O(n²) / O(1)
Quick Sort
O(n log n) / O(log n)
Merge Sort
O(n log n) / O(n)
