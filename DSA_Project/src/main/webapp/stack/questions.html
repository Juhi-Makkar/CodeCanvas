📈 Stack Fundamentals

LIFO principle - Last In, First Out. Perfect for parentheses validation, monotonic stacks, next greater elements.

// C: Stack Implementation #define MAX 100 int stack[MAX]; int top = -1; // Push void push(int x) { if(top < MAX-1) stack[++top] = x; }

Key Properties

  • LIFO: Last In, First Out
  • O(1) Operations: push, pop, peek
  • Top Pointer: Always points to last element
  • Overflow/Underflow: Handle edge cases
