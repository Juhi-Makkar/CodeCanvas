What is an Array?

An array is a collection of elements of the same data type stored in contiguous memory locations. It allows fast access using indexes.

Key Properties

  • Same data type
  • Contiguous memory
  • Fixed size
  • Fast access O(1)
  • Index starts from 0

Types of Arrays

  • 2D Array → Matrix

Frequently Asked Questions

An array is a fundamental data structure that stores a collection of elements of the same data type in contiguous memory locations. It allows efficient access to elements using an index, making it widely used in programming.
Arrays are faster because their elements are stored in contiguous memory locations, which allows direct access using indexing. This makes element retrieval very efficient with O(1) time complexity.
In most programming languages, arrays store elements of the same data type only. If mixed data types are needed, programmers usually use structures, objects, or lists instead.
The main limitation of arrays is their fixed size. Once declared, the size cannot be changed. If more space is needed, a new array must be created and elements copied.
