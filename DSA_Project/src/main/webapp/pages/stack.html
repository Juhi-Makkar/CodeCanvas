What is a Stack?

A stack is a linear data structure that follows the Last In, First Out (LIFO) principle. Elements are added and removed from the same end, called the top.

Key Properties

  • LIFO (Last In, First Out)
  • Operations: Push, Pop, Peek
  • Efficient O(1) operations
  • Used in recursion, undo mechanisms

Types of Stacks

  • Static Stack (fixed size)
  • Dynamic Stack (resizable)

Frequently Asked Questions

A stack is a linear data structure that follows the LIFO (Last In, First Out) principle, where elements are inserted and removed only from one end called the top.
LIFO stands for Last In, First Out, meaning the element that is added last to the stack will be removed first, similar to a stack of plates.
The main operations are: Push (add an element to the top), Pop (remove the top element), Peek (view the top element without removing it), and IsEmpty (check whether the stack is empty).
Push, Pop, and Peek operations all take O(1) time because they work only on the top element.
