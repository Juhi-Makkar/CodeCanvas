What is a Tree?

A tree is a hierarchical data structure consisting of nodes connected by edges. Each node has a parent-child relationship with exactly one root and no cycles.

Key Properties

  • Hierarchical structure
  • One root node
  • No cycles
  • Each node has 0+ children
  • Parent-child relationship

Types of Trees

  • Binary Tree → Max 2 children
  • Binary Search Tree → Sorted order
  • AVL Tree → Balanced BST
  • Heap → Complete tree

Frequently Asked Questions

A tree is a hierarchical data structure with nodes connected by edges, having one root and no cycles. Each node can have child nodes forming parent-child relationships.
A binary tree is a tree where each node has at most two children: left child and right child. It's the foundation for many advanced tree structures.
A BST is a binary tree where for each node, all elements in its left subtree are less than the node, and all in right subtree are greater.
Tree traversals are methods to visit all nodes: In-order (LNR), Pre-order (NLR), Post-order (LRN), and Level-order (BFS).
