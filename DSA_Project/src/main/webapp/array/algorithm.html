1. Array Initialization

Declare and initialize an array with fixed size.

Time: O(n) Space: O(n)

Pseudocode

PROCEDURE initializeArray(array, size, value)
    FOR i FROM 0 TO size-1 DO
        array[i] = value
    END FOR
END PROCEDURE

Implementation

C void initArray(int arr[], int size, int value) { for(int i = 0; i < size; i++) { arr[i] = value; } }

2. Array Traversal

Visit each element of the array sequentially.

Time: O(n) Space: O(1)

Algorithm Steps

1
Start from Index 0
Initialize index i = 0
2
Process Element
Visit arr[i] and perform operation
3
Increment Index
i = i + 1
4
Repeat
Continue until i < size

Implementation

C void traverseArray(int arr[], int size) { for(int i = 0; i < size; i++) { printf("%d ", arr[i]); } printf("\n"); }

3. Insert at End

Insert an element at the end of the array.

Time: O(1) Space: O(1)

Pseudocode

PROCEDURE insertEnd(array, size, element)
    array[size] = element
    size = size + 1
END PROCEDURE

Implementation

C void insertEnd(int arr[], int *size, int element) { if(*size < 100) { arr[(*size)++] = element; } }

4. Delete from End

Remove the last element from the array.

Time: O(1) Space: O(1)

Implementation

C int deleteEnd(int arr[], int *size) { if(*size > 0) { return arr[--(*size)]; } return -1; }

5. Linear Search

Search for an element by checking each element sequentially.

Time: O(n) Space: O(1) Worst: O(n) Best: O(1)

Algorithm Steps

1
Start from Beginning
Set i = 0
2
Compare Elements
If arr[i] == key, return i
3
Continue Search
Increment i and repeat
4
Not Found
Return -1 if end reached

Implementation

C int linearSearch(int arr[], int size, int key) { for(int i = 0; i < size; i++) { if(arr[i] == key) { return i; } } return -1; }

6. Binary Search

Efficient search on sorted array using divide and conquer.

Time: O(log n) Space: O(1)

Implementation

C int binarySearch(int arr[], int size, int key) { int left = 0, right = size - 1; while(left <= right) { int mid = left + (right - left) / 2; if(arr[mid] == key) return mid; else if(arr[mid] < key) left = mid + 1; else right = mid - 1; } return -1; }

7. Bubble Sort

Repeatedly swap adjacent elements if they are in wrong order.

Time: O(n²) Space: O(1)

Implementation

C void bubbleSort(int arr[], int n) { for(int i = 0; i < n-1; i++) { for(int j = 0; j < n-i-1; j++) { if(arr[j] > arr[j+1]) { int temp = arr[j]; arr[j] = arr[j+1]; arr[j+1] = temp; } } } }

8. Selection Sort

Select minimum element from unsorted portion and place at beginning.

Time: O(n²) Space: O(1)

Implementation

C void selectionSort(int arr[], int n) { for(int i = 0; i < n-1; i++) { int min_idx = i; for(int j = i+1; j < n; j++) { if(arr[j] < arr[min_idx]) min_idx = j; } int temp = arr[min_idx]; arr[min_idx] = arr[i]; arr[i] = temp; } }

9. Insertion Sort

Build sorted portion by inserting elements one by one.

Time: O(n²) Space: O(1)

Implementation

C void insertionSort(int arr[], int n) { for(int i = 1; i < n; i++) { int key = arr[i]; int j = i - 1; while(j >= 0 && arr[j] > key) { arr[j+1] = arr[j]; j--; } arr[j+1] = key; } }

10. Insert at Position

Insert element at specific position (requires shifting).

Time: O(n) Space: O(1)

Implementation

C int insertAtPos(int arr[], int *size, int pos, int element) { if(*size < 100 && pos >= 0 && pos <= *size) { for(int i = *size; i > pos; i--) { arr[i] = arr[i-1]; } arr[pos] = element; (*size)++; return 1; } return 0; }

11. Delete at Position

Delete element at specific position (requires shifting).

Time: O(n) Space: O(1)

Implementation

C int deleteAtPos(int arr[], int *size, int pos) { if(pos >= 0 && pos < *size) { int deleted = arr[pos]; for(int i = pos; i < *size - 1; i++) { arr[i] = arr[i+1]; } (*size)--; return deleted; } return -1; }

12. Find Maximum & Minimum

Find largest and smallest elements in array.

Time: O(n) Space: O(1)

Implementation

C void findMaxMin(int arr[], int size, int *max, int *min) { *max = *min = arr[0]; for(int i = 1; i < size; i++) { if(arr[i] > *max) *max = arr[i]; if(arr[i] < *min) *min = arr[i]; } }

13. Reverse Array

Reverse the elements of an array in-place.

Time: O(n/2) Space: O(1)

Implementation

C void reverseArray(int arr[], int size) { for(int i = 0; i < size / 2; i++) { int temp = arr[i]; arr[i] = arr[size - i - 1]; arr[size - i - 1] = temp; } }
