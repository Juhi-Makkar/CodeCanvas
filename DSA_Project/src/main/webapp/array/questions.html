📊 Array Fundamentals

Master the most fundamental data structure - Arrays. O(1) random access, contiguous memory, fixed size.

// C: Array Declaration int arr[5] = {10, 20, 30, 40, 50}; int* ptr = arr; // Array decays to pointer

Key Properties

  • O(1) Random Access: arr[i] = O(1)
  • Contiguous Memory: Cache-friendly
  • Fixed Size: Must know size upfront
  • Zero-based Indexing: arr[0] is first element
