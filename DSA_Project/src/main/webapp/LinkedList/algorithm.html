Linked List Traversal

Visit every node in the linked list exactly once in sequential order.

Time: O(n) Space: O(1)

Algorithm Steps

1
Initialize temp = head
Create temporary pointer pointing to head
2
While temp != NULL
Continue traversal until end of list
3
Process data
Print/access current node data
4
temp = temp->next
Move to next node

Implementation

C void traverseList(struct Node* head) { struct Node* temp = head; while(temp != NULL) { printf("%d -> ", temp->data); temp = temp->next; } printf("NULL\n"); }

Insert at Front

Insert new node at the beginning of the linked list (most efficient insertion).

Best: O(1) Space: O(1)

Algorithm Steps

1
Create new node
Allocate memory, set data value
2
new_node->next = head
Link new node to current head
3
head = new_node
Update head to point to new node

Implementation

C void insertAtFront(struct Node** head_ref, int new_data) { // 1. Allocate new node struct Node* new_node = (struct Node*) malloc(sizeof(struct Node)); // 2. Put data in new node new_node->data = new_data; // 3. Make next of new node as head new_node->next = (*head_ref); // 4. Move head to point to new node (*head_ref) = new_node; }

Delete by Value

Remove first occurrence of node with given value from linked list.

Worst: O(n) Space: O(1)

Algorithm Steps

1
Check head node
If head contains key to delete
2
Search for key
Traverse while keeping track of previous node
3
Unlink node
prev->next = temp->next
4
Free memory
Deallocate deleted node

Implementation

C void deleteNode(struct Node** head_ref, int key) { struct Node* temp = *head_ref; struct Node* prev = NULL; // If head node holds key if (temp != NULL && temp->data == key) { *head_ref = temp->next; free(temp); return; } // Search for key while (temp != NULL && temp->data != key) { prev = temp; temp = temp->next; } // If key not found if (temp == NULL) return; // Unlink node prev->next = temp->next; free(temp); }

Linear Search

Find position of target value in linked list (linear time complexity).

Time: O(n) Space: O(1)

Algorithm Steps

1
current = head, pos = 0
Initialize pointers and position counter
2
While current != NULL
Traverse through list
3
If match found
Return current position
4
Move forward
current = current->next, pos++

Implementation

C int search(struct Node* head, int key) { struct Node* current = head; int pos = 0; while (current != NULL) { if (current->data == key) { return pos; } current = current->next; pos++; } return -1; // Not found }

Reverse Linked List

Reverse direction of all pointers (iterative approach, O(1) space).

Time: O(n) Space: O(1)

Algorithm Steps

1
prev = NULL, current = head
Initialize three pointers
2
next = current->next
Save next node before overwriting
3
current->next = prev
Reverse current pointer
4
Move pointers forward
prev = current, current = next

Implementation

C void reverseList(struct Node** head_ref) { struct Node* prev = NULL; struct Node* current = *head_ref; struct Node* next = NULL; while (current != NULL) { next = current->next; current->next = prev; prev = current; current = next; } *head_ref = prev; }

Floyd's Cycle Detection

Tortoise and Hare algorithm to detect cycles in O(n) time, O(1) space.

Time: O(n) Space: O(1)

Algorithm Steps

1
slow = fast = head
Two pointers at same speed initially
2
slow += 1, fast += 2
Different speeds create relative movement
3
If slow == fast
Cycle detected (they meet in cycle)

Implementation

C int hasCycle(struct Node* head) { struct Node* slow = head; struct Node* fast = head; while (fast != NULL && fast->next != NULL) { slow = slow->next; fast = fast->next->next; if (slow == fast) { return 1; // Cycle exists } } return 0; // No cycle }
