# 🚀 CodeCanvas - Interactive DSA Learning Platform

CodeCanvas is an interactive Data Structures and Algorithms learning platform designed to make DSA concepts easy, visual, and engaging.

🌐 **Live Website:** https://codeecanvas.netlify.app/

---

## 📖 About the Project

CodeCanvas helps students understand Data Structures and Algorithms through visual explanations and organized topic sections. The platform focuses on making learning intuitive and beginner-friendly.

---

## ✨ Features

* 📚 Structured DSA learning
* 🔍 Search functionality
* 🎨 Clean and responsive UI
* 📱 Mobile-friendly design
* 📊 Visual representation of concepts
* ⚡ Fast and lightweight

---

## 📂 Topics Covered

* Arrays
* Linked Lists
* Stacks
* Queues
* Searching Algorithms
* Sorting Algorithms
